## Lestanianizer
ウェブサイトのフォントをレスタニア文字に置き換える Google Chrome 拡張機能を作りました。

* [ダウンロード](https://chrome.google.com/webstore/detail/lestanianizer/nkkckileddninpdecpogmmokegpkpijn)
